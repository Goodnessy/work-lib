# Work-Lib V1.5 内容补全与问 AI 版

本版本重点：

- 首页左上角为 G；搜索入口改为右上角单独的 🔍 图标。
- 首页“本周入库”去掉半月刊统计，保留 4 个对称数据块。
- 核心沉淀区域小字控制在两行以内。
- 底部标签固定为：员工活动｜公众号选题｜EAP｜AI 落地｜内部传播｜活动复盘。
- 补全文章正文，避免“点进去只有框架”。
- 最近一个月内容重点强化：公司家庭日、人力资源智能机器人。
- 文章详情页支持：收藏、划线批注、导出批注、选中文字后“问 AI”。
- “问 AI”不会接 API，也不会指定 ChatGPT 或 Gemini；它会复制一段通用提问，可粘贴到浏览器 AI 侧栏或任意 AI 对话框。

## 上传方式

1. 解压本 zip。
2. 打开 GitHub 仓库 `Goodnessy/work-lib`。
3. 上传解压后的所有文件到仓库根目录覆盖旧文件。
4. Commit changes。
5. 打开 `https://goodnessy.github.io/work-lib/?v=15` 强制刷新缓存。

## 注意

批注和收藏保存在当前浏览器 localStorage 中。换设备、换浏览器或清缓存可能丢失，请定期点击“导出批注”备份。
