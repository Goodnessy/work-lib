# Work-Lib V1.5.3 周更包｜阅读体验修正版

本包可直接上传到 GitHub Pages 仓库根目录覆盖旧文件。

## 本次更新

- 加入 2026-07-06 周一 Daily Brief，周一 Brief 已纳入本周周更包。
- 首页不调整视觉、结构和版式，仅修复标签可点击。
- 底部固定标签与正文标签支持点击，进入标签聚合 / 搜索结果。
- 栏目页默认显示最近更新，按时间倒序。
- 新增“按主题 / 按月份 / 按类型”辅助筛选，但默认折叠，不抢首屏。
- 文档索引不再每层默认展开，减少套娃感。
- 文章详情页辅助目录默认折叠，正文优先。
- 选中文字后的操作改为固定操作条，避免与浏览器原生菜单重叠。
- “问 AI”模板改为标题 + 当前链接 + 划线原文，用于解释和追问答案，不默认做工作转化。

## 上传方式

解压后，将所有文件上传到 `Goodnessy/work-lib` 仓库根目录覆盖旧文件。不要上传外层文件夹。

更新后访问：

```
https://goodnessy.github.io/work-lib/?v=153
```
