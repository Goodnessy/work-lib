# Work-Lib V1.2

浅蓝杂志风成熟内容版。

## 核心产品

- 每日简报
- 半月刊
- 灵感库
- 方法论
- 搜索

## 文件说明

- `index.html`：首页
- `daily.html`：每日简报列表
- `issue.html`：单期详情
- `magazine.html`：半月刊
- `inspiration.html`：灵感库
- `methods.html`：方法论
- `search.html`：搜索
- `data.js`：内容数据
- `style.css`：视觉样式
- `script.js`：交互逻辑


> 注：`collection.html` 保留为自动跳转页，兼容旧链接。
