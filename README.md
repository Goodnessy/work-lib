# Work-Lib

温和的工作知识库 V1.0。

这是一个适合部署在 GitHub Pages 上的静态网站，用于长期沉淀：

- 每日 Brief
- 深度推荐
- 生物医药行业动态
- 全球热点观察
- EAP / 员工活动 / 企业文化案例
- 宣传设计与视觉灵感
- AI 工具与效率方法
- 个人项目推进记录

## 文件说明

- `index.html`：网站首页
- `style.css`：视觉样式
- `script.js`：基础交互
- `README.md`：项目说明

## 推荐仓库名

建议仓库名使用小写：

`work-lib`

这样最终网址更稳定：

`https://你的GitHub用户名.github.io/work-lib/`

## 发布方式

建议使用 GitHub Pages：

1. Settings
2. Pages
3. Source: Deploy from a branch
4. Branch: main
5. Folder: /(root)
6. Save

## 注意

第一版请勿放入公司内部敏感资料、未公开项目、真实员工信息或内部文件截图。
