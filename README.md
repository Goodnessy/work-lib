# Work-Lib V1.3 文档库版

更新日期：2026-07-05

本版本保持首页浅蓝杂志风，内页改为更适合长期使用的文档库 / 文件树 / Markdown 阅读体验。

## 页面

- `index.html`：首页，本周主读 + 本周入库卡片
- `daily.html`：每日简报，多日内容归档
- `magazine.html`：半月刊，多期内容
- `inspiration.html`：灵感库
- `methods.html`：方法论
- `management.html`：管理视角
- `search.html`：搜索
- `issue.html`：通用详情页
- `data.js`：所有内容数据

## 上传 GitHub

进入仓库根目录，上传本文件夹内所有文件，覆盖同名文件后 Commit。不要上传外层文件夹本身。

## 内容说明

本包含多日、多期示例内容，用于确认信息结构和展示体验。新闻类内容基于公开来源整理；管理视角仅基于公开资料，不猜测未公开信息。
