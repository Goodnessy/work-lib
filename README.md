# Work-Lib V1.5.7｜更新助手与管理视角修正版

数据截止：2026-07-06

## 本版更新

1. 新增 `admin.html` 更新助手。
   - 首页底部只显示一个很淡的「更新助手」入口。
   - 更新助手只保留三个按钮：打开对话、上传更新、全量备份。
   - 三个按钮全部使用新标签打开，不覆盖当前 Work-Lib 页面。
   - 当前「打开对话」临时指向 ChatGPT 首页；用户提供当前对话 URL 后可替换为直达链接。
   - 当前「上传更新」临时指向 `https://github.com/Goodnessy/work-lib/upload/main`；用户提供最终上传 URL 后可替换。

2. 管理视角保持三大点不变：资料 / 动态 / 发言。
   - 不新增总览、素材库、话术库、案例库等一级模块。
   - 资料：补入公开资料边界、业务背景和检索状态。
   - 动态：补入 2026-03-03 正大天晴 × Sanofi rovadicitinib 全球授权合作，以及 2026-05-11 GSK / 中国生物制药 / 正大天晴 bepirovirsen 中国大陆供应合作。
   - 发言：如实记录截至 2026-07-06 暂未检索到可完整引用的 2026 年公开发言全文，并补入表达拆解，明确不是高管原话。

3. 继续保留并检查：标签聚合、栏目内索引、目录默认折叠、问 AI 简化模板、固定批注操作条。

## 上传方式

1. 更新前先打开 `admin.html`，点击「全量备份」，下载当前线上版本。
2. 从 ChatGPT 对话中下载本次新包。
3. 解压新包。
4. 点击「上传更新」，打开 GitHub 上传页面。
5. 上传解压后的全部文件，提交 Commit changes。

GitHub 不会自动解压 zip，请不要只上传 zip 文件。
