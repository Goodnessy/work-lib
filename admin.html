推荐操作顺序

  1. 点击「打开对话」，在 ChatGPT 对话里下载新生成的全量包。
  2. 点击「全量备份」，先保存当前线上版本，作为更新前保险。
  3. 解压新包，不要直接上传 zip。
  4. 点击「上传更新」，把解压后的全部文件拖入 GitHub 上传页。
  5. 提交 Commit changes，稍等 GitHub Pages 自动发布。

提醒：GitHub 不会自动解压 zip。当前「打开对话」先临时指向 ChatGPT 首页；你把本对话链接发来后，我会替换成直达当前对话。

本页不做什么

不做登录、不接 GitHub Token、不自动写仓库，也不再放“最新版本”按钮。纯静态网站无法自动获取 ChatGPT 刚生成但尚未上传的新包；所谓“最新版本”如果指当前线上包，会和「全量备份」重复。
